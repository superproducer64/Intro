Intro — iOS Build Files

Download both files, then double-click the installer.

1
Download the project files
The iOS Xcode project (~88KB)
Download intro-ios.tar.gz
2
Download the installer script
Extracts the project, installs dependencies, and opens Xcode automatically.
Download intro-build.command
3
Run the installer
Open Finder → Downloads. Right-click intro-build.command → Open → Open. A window runs everything and launches Xcode.
4
Archive in Xcode
In Xcode: select your team in Signing & Capabilities, then Product → Archive → Distribute App → App Store Connect.
Both files must be in your Downloads folder before running the installer.
